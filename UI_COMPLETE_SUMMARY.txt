================================================================================
                          TROUBLESHOOTING
================================================================================

PROBLEM: "ModuleNotFoundError: No module named 'tkinter'"
SOLUTION:
  - Tkinter is usually included with Python
  - If missing: pip install tk
  - Or reinstall Python with Tkinter option

PROBLEM: Panel doesn't appear
SOLUTION:
  1. Check if enabled in config:
     ui:
       enable_slideup_panel: true
  
  2. Check logs for errors:
     grep -i "ui\|panel" logs/item_assistant.log
  
  3. Try disabling and re-enabling UI

PROBLEM: Panel appears but doesn't update
SOLUTION:
  1. Verify wake word is working
  2. Check if STT is working
  3. Restart assistant

PROBLEM: Mic button doesn't work
SOLUTION:
  1. Verify Picovoice API key is set
  2. Check microphone permissions
  3. Test microphone with: python -c "import sounddevice"

PROBLEM: Panel animation is jerky
SOLUTION:
  1. Close other applications to free CPU
  2. Increase animation speed:
     ANIMATION_SPEED = 10  # Lower = faster
     ANIMATION_STEP = 25   # Higher = faster

PROBLEM: Assistant works but UI crashes
SOLUTION:
  1. Disable UI:
     ui:
       enable_slideup_panel: false
  
  2. Check error logs
  3. Update tkinter: pip install --upgrade tk
